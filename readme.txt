Hotel Booking System

Welcome to the Hotel Booking System, a web application that facilitates hotel room reservations. Users can register, log in, and manage their bookings, while administrators can oversee the system's operations. The system also integrates PayPal for payment processing.
Getting Started

    Register your account on the website.
    Log in using your registered credentials.
    Browse available rooms, view details, and make reservations.
    For administrators, log in to the admin panel to manage users, hotels, rooms, and bookings.

Features
Rooms.php

    Displays a list of available rooms and their details.
    Users can view room availability and pricing.
    Allows users to make reservations and select check-in/check-out dates.

Rooms-single.php

    Provides detailed information about a specific room.
    Enables users to view room images and facilities.
    Handles check-in and checkout dates using the Bootstrap datepicker.

User Accounts

    Users can register and log in to manage their reservations.
    Administrators have access to the admin panel to manage system settings.

Setup Instructions

    Clone this repository to your local machine.
    Set up your local development environment (e.g., XAMPP, WAMP, etc.).
    Import the provided MySQL database file to set up the required database tables.
    Modify the database configuration in the necessary files to match your local environment.
    Access the application using your preferred web browser.
